About Project

Fake Advertisement Detection Using ML and Data Science

This project is designed to identify fake and misleading online advertisements. It uses text analysis and machine learning methods to classify advertisements, predict risk score, group similar advertisements, and recommend safety actions. The system is useful for identifying scam offers, fake loan ads, prize frauds, misleading job advertisements, and suspicious online promotions.
